# CSS选择器优先级

| CSS选择器                | 优先级 |
| ------------------------ | ------ |
| important                |        |
| 内联                     | 1000   |
| id                       | 100    |
| class、伪类、属性选择器  | 10     |
| 元素选择器、伪元素选择器 | 1      |

Id > class > 元素选择器 > 兄弟选择器 > 子代选择器 > 后代选择器 > 属性选择器 > 伪元素 伪类

`ID选择器` 比如#header

`类选择器` 比如.promo

`元素选择器` 比如 div

`兄弟选择器` 比如 h2 + p

`子选择器` 比如 li  >  ul

`后代选择器` 比如 ul a 7. 通用选择器 比如 *

`属性选择器` 比如 type = “text”

`伪类/伪元素选择器` 比如 a:hover
