# 正则表达式

正则表达式是一种强大的文本匹配工具，本教程从入门到进阶覆盖常用场景。

## 目录

- [正则入门](./入门) - 基本语法、常用模式

- [正则进阶](./正则进阶) - 分组、零宽断言、Python re 库

## 快速入门

```python
import re

# 查找手机号
text = "我的手机号是138-1234-5678"
pattern = r"1[3-9]\d{1}-\d{4}-\d{4}"
match = re.search(pattern, text)
print(match.group())  # 138-1234-5678
```
