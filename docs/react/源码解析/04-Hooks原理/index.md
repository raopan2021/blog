# 模块四：Hooks 原理

> 深入理解 useState、useEffect 等 Hook 的实现原理

---

## 📖 本模块总结

Hooks 是 React 16.8 引入的新特性，让函数组件也能拥有状态和生命周期。

| Hook | 说明 |
|------|------|
| useState | 状态管理 |
| useReducer | 基于 reducer 的状态管理 |
| useEffect | 副作用处理 |
| useRef | 引用 DOM 或保存值 |
