## 📖 本模块总结

React 的核心概念包括：**JSX 编译**、**Virtual DOM**、**React Element**。

| 概念 | 说明 |
|------|------|
| JSX | 语法糖，编译为 React.createElement |
| Virtual DOM | JS 对象描述 UI |
| React Element | Virtual DOM 的基本单位 |
