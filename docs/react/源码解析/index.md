# React 源码解析

> 从零开始，深入理解 React 的核心实现原理

---

## 📚 课程大纲

### 模块一：React 基础概念

- JSX 编译原理
- Virtual DOM 结构

### 模块二：React 16 Fiber 架构

- Fiber 数据结构
- 双缓冲与工作循环

### 模块三：调和算法

- Diff 算法原理
- key 的作用

### 模块四：Hooks 原理

- useState 实现原理
- useEffect 实现原理
- useRef 与性能优化

### 模块五：状态管理

- Redux 核心原理
- Zustand 轻量方案

---

## 🎯 学习目标

1. **理解 Fiber 架构** — 了解 React 16 的核心改动
2. **掌握调和算法** — Diff 算法的原理
3. **理解 Hooks 机制** — 状态 Hook 的实现
4. **阅读源码能力** — 遇到问题能追根溯源
