## 🎯 学习目标

1. **理解 Vue3 的核心响应式系统** — 掌握 Proxy、Effect、Watch 的原理
2. **了解组件挂载与更新流程** — 从 template 到真实 DOM 的转变
3. **掌握性能优化思路** — 虚拟 DOM、懒加载、computed 缓存
4. **具备阅读源码的能力** — 遇到问题能追根溯源
