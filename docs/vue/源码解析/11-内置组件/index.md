# 模块十一：内置组件

> 深入理解 Vue 内置组件：KeepAlive、Transition、TransitionGroup

## 📖 本模块总结

Vue 提供了一系列内置组件，提供特殊的 DOM 操作能力。

| 组件 | 说明 |
|------|------|
| KeepAlive | 缓存组件实例，避免重复挂载/卸载 |
| Transition | 单元素动画过渡 |
| TransitionGroup | 列表动画过渡 |
| Teleport | 传送到指定 DOM 位置 |
| Suspense | 异步组件加载状态 |
