# 模块五：性能优化

## 知识点预览

- **虚拟 DOM** — 用 JS 对象描述真实 DOM，跨平台更方便
- **Diff 算法** — 比较新旧虚拟 DOM，找出最小更新量
- **patch** — 将变化应用到真实 DOM
- **编译时优化** — static props hoisting、cache handler 等
