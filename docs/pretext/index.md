# Pretext 文本布局

> 不触碰 DOM 的精确文本测量与多行布局

---

## 📚 课程大纲

### 模块一：基础

- Pretext 快速入门

### 模块二：实现原理

- Canvas 文本测量原理
- 字形分段与 Unicode 处理
- 换行算法与粘合规则

### 模块三：进阶

- API 详解与高级用法
- 实战技巧与性能优化

### 模块四：Demo 项目

- Textarea 自动高度
- Canvas 文本渲染器
- 虚拟列表实现

---

## 🎯 学习目标

1. **理解 Pretext 解决的问题** — DOM 回流与文本测量
2. **掌握核心 API** — prepare / layout / layoutWithLines
3. **理解实现原理** — Canvas + Unicode 分段
4. **具备实战能力** — 虚拟滚动、文本环绕等场景应用
