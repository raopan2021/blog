## Demo 1：Textarea 自动高度

**功能**：用户在 Textarea 中输入文字，框自动调整高度，无需 DOM 测量。

**核心代码**：

```typescript
import { prepare, layout } from '@chenglou/pretext'

let prepared = null
let lastText = ''

function update() {
  const value = textarea.value
  const width = textarea.clientWidth - padding
  const font = `${fontSize} ${fontFamily}`
  const lineHeight = parseFloat(getComputedStyle(textarea).lineHeight)

  // 文本变了才重新 prepare
  if (value !== lastText) {
    lastText = value
    prepared = prepare(value, font, { whiteSpace: 'pre-wrap' })
  }

  const { height } = layout(prepared, width, lineHeight)
  textarea.style.height = Math.max(height + padding, minHeight) + 'px'
}

textarea.addEventListener('input', update)
```

**关键点**：
- `whiteSpace: 'pre-wrap'` 保留原始换行和空格
- 文本变化时才重新 `prepare`，resize 时只调 `layout`
- 等待 `document.fonts.ready` 确保字体加载完成

## Demo 2：Canvas 文本渲染

**功能**：在 Canvas 上渲染多行文本，不创建任何 DOM 元素。

**核心代码**：

```typescript
import { prepareWithSegments, layoutWithLines } from '@chenglou/pretext'

const prepared = prepareWithSegments(text, font)
const { lines } = layoutWithLines(prepared, maxWidth, lineHeight)

ctx.font = font
lines.forEach((line, i) => {
  ctx.fillText(line.text, x, y + i * lineHeight)
})
```

**关键点**：
- `layoutWithLines` 返回每行的文本内容
- 适合游戏、教育图形、数据可视化等 Canvas 场景
- 自动处理混合语言、Emoji、多方向文本

## Demo 3：精准高度虚拟列表

**功能**：虚拟列表每行高度由文本真实长度决定，滚动条完全准确。

**核心代码**：

```typescript
// 预处理所有 item
const prepared = items.map(item => prepare(item.text, font))

// 获取真实高度
function getHeight(id, width) {
  const { height } = layout(prepared[id], width, lineHeight)
  return height + padding
}

// 计算总高度
let totalH = 0
const offsets = []
for (const item of items) {
  offsets.push(totalH)
  totalH += getHeight(item.id, width)
}
container.style.height = totalH + 'px' // 关键：设置总高度让滚动条准确
```

**关键点**：
- 传统虚拟列表用估算高度，滚动条会跳
- Pretext 计算真实高度 → 滚动条始终准确
- 只渲染可视区域 + 少量缓冲区的 DOM 节点

## Demo 4：文本环绕浮动元素

**功能**：文字自动绕着左侧图片流动，图片下方文字使用完整宽度。

**核心代码**：

```typescript
import { prepareWithSegments, layoutNextLine } from '@chenglou/pretext'

const prepared = prepareWithSegments(text, font)
let cursor = { segmentIndex: 0, graphemeIndex: 0 }
let y = 0

while (true) {
  // 浮动区域内的行更窄
  const inFloatZone = y < floatBottom
  const leftMargin = inFloatZone ? floatWidth + gap : 0
  const width = maxWidth - leftMargin

  const line = layoutNextLine(prepared, cursor, width)
  if (!line) break

  ctx.fillText(line.text, leftMargin, y)
  cursor = line.end.cursor
  y += lineHeight
}
```

**关键点**：
- `layoutNextLine` 是流式的，每次只计算一行
- 每行宽度可以动态变化（不像其他 API 需要固定宽度）
- 适合图文混排、报表、数据可视化
