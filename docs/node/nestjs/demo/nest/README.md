## 安装依赖

```bash
pnpm install
```

## 运行

```bash
# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:prod
```

## 项目目录

```txt
| --- src   源码
    | --- upload-pic  上传下载图片模块
    | --- user 模块

```
