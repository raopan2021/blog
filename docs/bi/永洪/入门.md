### 1.1 下载安装

永洪 BI 提供多版本下载，推荐从 Desktop 免费版开始：

1. 访问官网下载页：[https://www.yonghongtech.com/downloads](https://www.yonghongtech.com/downloads)
2. 选择 **Desktop 版本**（永久免费）

> 💡 永洪 Desktop 功能完整，适合个人学习和小型项目，无数据量限制。

## 二、添加数据源

### 2.1 连接数据库

永洪 BI 支持多种数据库连接（MySQL、Oracle、SQL Server、PostgreSQL 等）：

**操作步骤：**

1. 切换到「数据源」界面
2. 点击「+」新建数据源
3. 选择数据库类型（如 MySQL）
4. 填写连接信息

```
┌────────────────────────────────────────────┐
│  新建数据源                                │
├────────────────────────────────────────────┤
│  数据源类型：MySQL                    [▼]  │
│                                            │
│  驱动：com.mysql.jdbc.Driver               │
│  URL：jdbc:mysql://localhost:3306/testdb  │
│  用户名：root                               │
│  密码：******                               │
│                                            │
│  服务器地址：127.0.0.1                      │
│  端口：3306                                 │
│  数据库：testdb                             │
│                                            │
│  [测试连接]            [保存]              │
└────────────────────────────────────────────┘
```

**常用数据库连接示例：**

```sql
-- MySQL
jdbc:mysql://192.168.1.100:3306/bi_db?useUnicode=true&characterEncoding=UTF-8

-- PostgreSQL
jdbc:postgresql://192.168.1.100:5432/bi_db

-- Oracle
jdbc:oracle:thin:@192.168.1.100:1521:orcl

-- SQL Server
jdbc:sqlserver://192.168.1.100:1433;DatabaseName=bi_db
```

### 2.2 本地上传文件

支持上传 Excel、CSV 等本地文件：

1. 点击「+」→「EXCEL&TEXT」
2. 选择本地文件（.xlsx / .xls / .csv）
3. 选择需要导入的 Sheet 页
4. 点击「刷新数据」→「保存」

> 💡 如果出现乱码，先将文件编码改为 **UTF-8** 再上传。

## 三、创建数据集

数据集是报表开发的核心，承上启下：向上承接数据源，向下为报告提供数据。

### 3.1 数据集类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| **SQL 数据集** | 直接编写 SQL 查询 | 有数据库，灵活查询 |
| **Excel 数据集** | 上传本地 Excel | 简单数据，小数据量 |
| **内嵌数据集** | 在 BI 中手动录入 | 临时数据、测试数据 |
| **组合数据集** | 多数据源 Join/Union | 跨库数据整合 |

### 3.2 创建 SQL 数据集

1. 切换到「数据集」界面
2. 点击「新建数据集」→「SQL 数据集」
3. 选择数据源
4. 编写 SQL 语句

```sql
-- 示例：销售数据查询
SELECT 
    s.order_date   AS "订单日期",
    p.province     AS "省份",
    c.category_name AS "商品类别",
    SUM(s.amount)  AS "销售额",
    COUNT(s.order_id) AS "订单数"
FROM sales s
LEFT JOIN product p ON s.product_id = p.id
LEFT JOIN category c ON p.category_id = c.id
WHERE s.order_date >= '2024-01-01'
GROUP BY s.order_date, p.province, c.category_name
ORDER BY s.order_date, SUM(s.amount) DESC
```

### 3.3 数据集参数

当数据量较大时，可以通过参数控制数据范围：

**方法一：使用过滤器添加参数**

```sql
-- SQL 中使用参数
SELECT * FROM sales 
WHERE region = '${selectRegion}'
  AND order_date >= '${startDate}'
  AND order_date <= '${endDate}'
```

**方法二：SQL 语句中直接写参数**

```sql
-- 日期参数（可选）
SELECT * FROM sales 
WHERE 1=1
${if(len(startDate) > 0, "AND order_date >= '" + startDate + "'", "")}
${if(len(endDate) > 0, "AND order_date <= '" + endDate + "'", "")}
```

### 3.4 组合数据集

组合数据集可以对多个数据集进行 Join 或 Union：

```
数据集A (销售表)          数据集B (产品表)
┌──────────────┐           ┌──────────────┐
│ product_id   │──Join───→│ id           │
│ amount       │           │ product_name │
│ order_date   │           │ category     │
└──────────────┘           └──────────────┘
```

## 五、参数与过滤

### 5.1 添加参数控件

参数控件让用户可以动态筛选数据：

**步骤：**

1. 在报告顶部拖入「文本下拉框」或「日期选择器」等控件
2. 绑定参数名称（如 `selectRegion`）
3. 设置控件数据来源（可以是固定值或数据集）

```
┌─────────────────────────────────────────────────┐
│  省份：[  全部 ▼  ]  日期：[2024-01 ─ 2024-12]  │
│                          [查询]  [重置]          │
└─────────────────────────────────────────────────┘
```

### 5.2 全局过滤器

报告级全局过滤，对所有组件同时生效：

1. 点击「报告」→「全局滤镜」
2. 添加过滤条件
3. 绑定参数控件

### 5.3 组件过滤器

单个组件的独立过滤，不影响其他组件。

## 六、组件交互

### 6.1 钻取（Drill）

点击数据进入下一层级明细：

```

华北 → 北京市 → 朝阳区
```

**设置方式：** 组件属性 → 交互 → 添加钻取关系

- 点击地图上某个省份 → 右侧柱状图只显示该省数据
- 点击柱状图某个月份 → 下方表格只显示该月明细

**设置方式：** 报告工具栏 → 更多 → 组件传参

### 6.3 链接跳转


## 七、自由式表格

自由式表格是制作复杂中国式报表的核心组件：

### 7.1 特点

- 支持多数据源关联
- 灵活定义单元格计算公式
- 类似 Excel 的操作体验
- 支持行列合并、格间计算

### 7.2 常用公式

```javascript
// 求和
=SUM(E2:E100)

// 平均值
=AVG(E2:E100)

// 条件求和（计算华南区销售额）
=SUMIF(C2:C100, "华南", E2:E100)

// 同比增长率
=(E3-E2)/E2 * 100

// 排名
=RANK(E2, E$2:E$100)

// 占比
=E2/SUM(E$2:E$100) * 100
```

## 十、实战示例：销售看板

### 10.1 需求

制作一个年度销售分析看板，包含：

- 年度销售总额（指标卡）
- 月度销售趋势（折线图）
- 各省份销售排名（柱状图）
- 产品类别占比（饼图）
- TOP10 客户表格（自由式表格）

### 10.2 数据集准备

```sql
-- 1. 月度销售汇总
SELECT 
    DATE_FORMAT(order_date, '%Y-%m') AS month,
    SUM(amount) AS total_amount,
    COUNT(DISTINCT customer_id) AS customer_count
FROM sales
WHERE YEAR(order_date) = ${selectYear}
GROUP BY DATE_FORMAT(order_date, '%Y-%m')

-- 2. 省份销售排名
SELECT province, SUM(amount) AS total
FROM sales
WHERE YEAR(order_date) = ${selectYear}
GROUP BY province
ORDER BY total DESC

-- 3. 类别占比
SELECT category, SUM(amount) AS total
FROM sales
WHERE YEAR(order_date) = ${selectYear}
GROUP BY category

-- 4. TOP10 客户
SELECT customer_name, SUM(amount) AS total
FROM sales
WHERE YEAR(order_date) = ${selectYear}
GROUP BY customer_name
ORDER BY total DESC
LIMIT 10
```

## 十一、常见问题

### 11.1 数据集连接失败

**检查项：**
- JDBC 驱动是否正确
- 数据库服务是否启动
- IP、端口、数据库名是否正确
- 用户名密码是否正确

### 11.2 数据量大了很慢

**优化方案：**
1. 勾选「数据库内计算」下推
2. 合理使用 WHERE 条件过滤数据
3. 对大表建立索引
4. 使用参数控制数据范围

### 11.3 报告导出失败

- 检查 PDF 插件是否安装
- 报告是否包含不支持导出的组件
