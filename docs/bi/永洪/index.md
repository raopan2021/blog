## 三、核心优势

### 1. 性能强劲
- 自研 **VooltDB** 高性能计算引擎
- 支持 **10 亿级数据**秒级响应
- 数据库内计算，数据不下沉

### 2. 功能全面
- 近 **60+ 可视化组件**
- 支持 ** ETL 数据处理**（Data Modeler）
- 内置 **AI 能力**（天枢问数、数据洞察）
- 数字孪生大屏，酷炫动态效果

### 3. 易于使用
- 拖拽式操作，零编码
- 业务人员也能快速上手
- 丰富的报表模板市场

### 4. 国产化适配
- 全面支持国产化操作系统、数据库
- 通过等保三级认证
- 适合政府、金融等敏感行业

## 六、报表开发流程

永洪 BI 的报表开发遵循以下流程：

```
1. 添加数据源    → 连接数据库 / 上传文件 / API 接口
       ↓
2. 创建数据集    → SQL 数据集 / Excel 数据集 / 组合数据集
       ↓
3. 数据建模      → Data Modeler 数据处理与关联
       ↓
4. 制作报告      → 拖拽组件，绑定维度和度量
       ↓
5. 配置交互      → 参数过滤、组件传参、钻取
       ↓
6. 发布与分享    → 门户发布 / 链接分享 / 权限配置
```
