# 帆软 FineReport 简介

## 一、产品概述

帆软软件（FanRuan Software）是国内最大的报表软件厂商，旗下产品 FineReport 和 FineBI 广泛应用于各行各业。据多个第三方报告显示，帆软连续多年位居国内 BI/报表市场占有率第一。

**官网**：[https://www.finereport.com](https://www.finereport.com)  
**社区**：[https://community.fanruan.com](https://community.fanruan.com)  
**帮助文档**：[https://help.fanruan.com/finereport](https://help.fanruan.com/finereport)

## 二、核心产品线

| 产品 | 定位 | 核心优势 |
|------|------|---------|
| **FineReport** | 报表制作 | 中国式复杂报表、ECharts 集成 |
| **FineBI** | 自助分析 | 业务人员自助、敏捷 BI |
| **FineDataLink** | 数据集成 | ETL / 数据管道 |

> 💡 **FineReport** 适合开发固定格式的复杂报表（如财务报表）；**FineBI** 适合业务人员自助探索分析。

## 三、FineReport 核心特点

### 3.1 报表能力强

- **中国式报表**：支持复杂表头、分组报表、交叉报表
- **填报报表**：支持数据录入和审批流程
- **决策报表**：多 sheet 联动、酷炫大屏
- **移动报表**：适配手机和平板

### 3.2 生态完善

- 内置 **ECharts** 图表库
- 支持第三方图表（如 Highcharts）
- 丰富的 **模板市场** 和案例库
- 强大的 **插件体系**

### 3.3 学习资源丰富

- 官方帮助文档详尽
- 社区活跃，问题响应快
- 大量视频教程和实战案例
- 持证工程师众多

## 四、与 FineBI 对比

| 对比项 | FineReport | FineBI |
|--------|-----------|--------|
| **主要定位** | 复杂报表、固定格式报表 | 自助分析、探索式分析 |
| **典型场景** | 财务报表、业务统计表 | 业务人员自助分析 |
| **学习门槛** | 需要一定 SQL 基础 | 业务人员可直接上手 |
| **灵活性** | 固定模板，格式严谨 | 拖拽分析，随时组合 |
| **可视化** | ⭐⭐⭐⭐ 强 | ⭐⭐⭐⭐⭐ 极强 |
| **复杂报表** | ⭐⭐⭐⭐⭐ 极强 | ⭐⭐⭐ 一般 |

## 五、学习路径

```
第一步：熟悉设计器
  └→ 安装 FineReport 设计器，熟悉界面布局

第二步：掌握报表基础
  └→ 单元格扩展、父格设置、数据绑定

第三步：学会 SQL 数据集
  └→ 连接数据库，编写 SQL 查询

第四步：参数与控件
  └→ 实现动态查询和页面过滤

第五步：图表制作
  └→ 掌握各种图表类型和数据绑定

第六步：实战项目
  └→ 从简单报表到复杂报表逐步提升
```
