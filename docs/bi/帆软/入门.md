# 帆软 FineReport 开发入门

## 一、环境准备

### 1.1 下载安装

帆软 FineReport 提供免费设计器下载：

1. 访问帆软官网：[www.finereport.com](https://www.finereport.com)
2. 下载 FineReport 设计器（帆软报表）
3. 解压后运行 `FineReport_xxx.exe`

> 设计器部分功能免费使用，企业级功能需要 license。

### 1.2 设计器目录结构

```
FineReport/
├── bin/                    # 启动脚本
│   ├── FineReport64.exe   # 64位启动程序
│   └── FineReport32.exe   # 32位启动程序
├── plugins/                # 插件目录
├── reportlets/              # 报表模板存放目录
├── cache/                   # 缓存文件
└── log/                     # 日志文件
```

## 二、基本概念

### 2.1 重要术语

| 术语 | 说明 |
|------|------|
| **模板 (.cpt)** | 帆软报表文件格式，类似于 Excel 文件 |
| **数据集** | SQL 查询结果或程序数据集，类似 Excel 数据源 |
| **单元格** | 报表基本单元，可存储数据、文本、公式 |
| **参数** | 用于筛选数据的变量，类似 SQL WHERE 条件 |
| **父子格** | 单元格的上下级关联关系 |

### 2.2 单元格扩展方向

- **默认**：纵向扩展，数据向下排列
- **横向扩展**：数据向右排列
- **列表/分组**：根据分组字段扩展

## 三、创建第一个报表

### 3.1 新建模板

1. 打开 FineReport 设计器
2. 点击「文件」→「新建模板」
3. 选择「空白报表」或「blank_simple.grf」

### 3.2 添加数据集

1. 打开「服务器」→「数据集」
2. 点击「新建数据集」
3. 选择数据连接（需要先配置数据连接）

```sql
-- 示例 SQL
SELECT 
    d.dept_name AS "部门",
    e.emp_name AS "姓名",
    e.salary AS "工资",
    e.entry_date AS "入职日期"
FROM employee e
LEFT JOIN department d ON e.dept_id = d.id
ORDER BY d.dept_name, e.salary DESC
```

### 3.3 设计报表

1. 将数据集字段拖入单元格
2. 设置表头（标题行）
3. 调整列宽、行高
4. 添加边框

```
   A        B         C         D         E
1  部门      姓名       工资       入职日期
2  =A2     =B2       =C2       =D2      (数据扩展)
3
```

### 3.4 设置父格

父格设置决定单元格的扩展行为：

- **左父格**：向右侧扩展
- **上父格**：向下扩展

```
设置前：所有单元格独立扩展
┌────┬────┬────┐
│ A1 │ A1 │ A1 │
├────┼────┼────┤
│ A2 │ A2 │ A2 │
└────┴────┴────┘

设置后：A1 为 B1、C1 的左父格
┌────┬────┬────┐
│ A1 │ B1 │ C1 │
├────┼────┼────┤
│ A2 │ B2 │ C2 │
└────┴────┴────┘
```

## 四、分组报表

### 4.1 简单分组

1. 选中数据单元格
2. 右键选择「分组」
3. 设置分组表达式

```javascript
// 分组表达式示例
// 按部门分组
=if(A2 != A1, 1, 0)

// 高级分组：使用数据集字段
=if(ds1.groupValue("dept_name", A2) != ds1.groupValue("dept_name", A1), 1, 0)
```

### 4.2 条件分组

```javascript
// 按工资区间分组
=if(${salary} < 5000, "5000以下", 
    if(${salary} < 10000, "5000-10000", "10000以上"))
```

## 五、参数控件

参数用于页面端筛选数据，类似 SQL WHERE 条件。

### 5.1 定义参数

1. 打开「模板」→「模板参数」
2. 新建参数，如 `dept_name`（部门名称）

### 5.2 使用参数

```sql
-- 在 SQL 数据集中使用参数
SELECT * FROM employee 
WHERE 1=1
${if(len(dept_name) > 0, "AND dept_name = '" + dept_name + "'", "")}
```

### 5.3 添加控件

1. 在参数面板添加控件
2. 控件类型：下拉框、文本框、日期选择器等
3. 绑定参数

```
┌────────────────────────────────────┐
│  请选择部门: [下拉框 ▼] [查询]     │
└────────────────────────────────────┘
```

### 5.4 常用控件类型

| 控件 | 适用场景 | 特点 |
|------|---------|------|
| 下拉框 | 单选固定值 | 简单易用 |
| 下拉复选框 | 多选固定值 | 支持多选 |
| 文本框 | 模糊搜索 | 灵活输入 |
| 日期选择器 | 日期筛选 | 选择日期 |
| 数字框 | 数值范围 | 数值类型 |

## 六、图表制作

### 6.1 添加图表

1. 点击菜单「插入」→「图表」
2. 选择图表类型
3. 绑定数据

### 6.2 常用图表类型

| 图表类型 | 适用场景 |
|---------|---------|
| 柱状图 | 对比分类数据大小 |
| 折线图 | 展示趋势变化 |
| 饼图 | 展示占比关系 |
| 散点图 | 两个变量关系 |
| 雷达图 | 多维度对比 |
| 地图 | 地域分布 |

### 6.3 图表数据绑定

```
系列名: 绑定分类轴（X轴）
系列值: 绑定数值（Y轴）
```

## 七、样式设置

### 7.1 单元格样式

- **字体**：字体类型、大小、颜色、加粗
- **对齐**：水平/垂直对齐
- **背景色**：奇偶行不同色
- **边框**：内外边框设置

### 7.2 条件属性

根据数据值动态改变样式：

```
条件1：salary > 10000 → 字体红色，加粗
条件2：salary < 5000  → 背景黄色
```

### 7.3 常用公式

```javascript
// 求和
=SUM(C2:C100)

// 平均值
=AVG(C2:C100)

// 计数
=COUNT(B2:B100)

// 条件计数
=COUNTIF(C2:C100, ">5000")

// 取整
=ROUND(C2, 2)  // 保留2位小数

// 日期差
=DATEDIF(D2, TODAY(), "D")  // 计算入职天数
```

## 八、实战示例：销售报表

### 8.1 需求

制作一张年度销售报表，包含：
- 月度销售额柱状图
- 地区销售排名表
- TOP10 客户表格

### 8.2 实现步骤

1. **创建数据集**（3个）

```sql
-- 数据集1：月度销售
SELECT month, SUM(amount) AS total
FROM sales
WHERE year = ${selectYear}
GROUP BY month
ORDER BY month

-- 数据集2：地区销售
SELECT region, SUM(amount) AS total
FROM sales
WHERE year = ${selectYear}
GROUP BY region
ORDER BY total DESC

-- 数据集3：TOP10客户
SELECT customer_name, SUM(amount) AS total
FROM sales
WHERE year = ${selectYear}
GROUP BY customer_name
ORDER BY total DESC
LIMIT 10
```

2. **设计模板布局**

```
┌──────────────────────────────────────────┐
│           📊 年度销售报表                  │
│           ${selectYear}年                 │
├──────────────────────────────────────────┤
│  [年份下拉框 ▼]  [查询]                   │
├──────────────────────────────────────────┤
│           【月度销售趋势图】               │
│                                          │
├──────────────────────────────────────────┤
│  【地区销售排名】    │  【TOP10客户】     │
│  ┌──────────────┐  │  ┌────────────┐   │
│  │ 地区   销售额 │  │  │ 客户  销售额│   │
│  │ 华南   1000万 │  │  │ AAA   500万 │   │
│  │ 华北   800万  │  │  │ BBB   400万 │   │
│  └──────────────┘  │  └────────────┘   │
└──────────────────────────────────────────┘
```

3. **添加参数控件**（年份下拉框）
4. **设置图表**（绑定月度销售数据）
5. **预览测试**

## 九、常见问题

### 9.1 数据集连接失败

检查：
1. JDBC 驱动是否正确放置
2. 数据库连接信息是否正确
3. 网络是否能连通数据库

### 9.2 报表显示乱码

解决：
1. 检查设计器编码设置
2. 统一使用 UTF-8 编码
3. 数据库连接串加编码参数

### 9.3 性能优化

- 避免 SELECT *
- 善用 WHERE 条件过滤数据
- 合理建立索引
- 大数据量考虑使用分组
