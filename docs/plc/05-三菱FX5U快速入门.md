# 05-三菱 FX5U 快速入门

## 5.1 软件与硬件准备

### 编程软件

三菱 PLC 使用 **GX Works3** 进行编程。

| 软件 | 支持系列 | 说明 |
|------|----------|------|
| **GX Works3** | FX5U / iQ-R 系列 | 推荐使用，界面现代 |
| GX Developer | FX 全系列 | 老版本，仅支持 FX 系列 |

下载地址：三菱电机官网（需注册账号）

### 硬件准备

| 物品 | 说明 |
|------|------|
| FX5U-32MT/ES 或 FX5U-64MT | 主流型号，晶体管输出 |
| 编程线缆（USB 或以太网） | FX5U 自带 Mini-USB 接口 |
| DC 24V 电源 | 为 PLC 供电 |
| 输入/输出负载 | 调试用 |

### FX5U 面板说明

```
┌──────────────────────────────────────┐
│  FX5U-32MT                            │
│  ⚡ RUN  ERROR  POWER                 │
│  ────────────────────────────────────│
│  输入: X0  X1  X2  ... X17           │
│  ────────────────────────────────────│
│  输出: Y0  Y1  Y2  ... Y17  (晶体管)  │
└──────────────────────────────────────┘
```

## 5.2 GX Works3 项目创建

### 步骤 1：新建工程

1. 打开 GX Works3 → 点击 **"工程"** → **"新建工程"**
2. 选择 **"系列"**：**FX5U**
3. 选择 **"机型"**：对应具体型号（如 `FX5U-32MT/ES`）
4. 选择 **"程序语言"**：梯形图（LD）或结构化文本（ST）
5. 点击 **"确定"**

### 步骤 2：工程树结构

```
工程
├──  Program Body
│    ├──  Program1 (MAIN)   ← 主程序
│    └──  副程序（可选）
├──  智能功能模块
├──  参数
│    ├──  PLC参数
│    └──  软元件参数
└──  备注
```

## 5.3 软元件（变量）体系

三菱 PLC 使用 **软元件** 作为变量，地址体系与西门子不同。

### 软元件一览

| 类型 | 标记 | 地址范围 | 说明 |
|------|------|----------|------|
| 输入继电器 | X | X0~X177 | 对应外部输入 |
| 输出继电器 | Y | Y0~Y177 | 对应外部输出 |
| 辅助继电器 | M | M0~M7999 | 内部中间继电器 |
| 锁存继电器 | L | L0~L2047 | 断电保持 |
| 定时器 | T | T0~T511 | 延时定时 |
| 计数器 | C | C0~C255 | 增/减计数 |
| 数据寄存器 | D | D0~D7999 | 存储数据，16bit |
| 链接继电器 | B | B0~BFFF | 网络通信用 |
| 链接寄存器 | W | W0~W1FFF | 网络数据 |

### 特殊继电器（常用）

| 编号 | 名称 | 用途 |
|------|------|------|
| M8000 | 常时ON | 运行中始终为1 |
| M8001 | 常时OFF | 运行中始终为0 |
| M8002 | 初始脉冲 | 程序开始第一个扫描周期为1 |
| M8011 | 10ms 时钟 | 10ms 周期脉冲 |
| M8012 | 100ms 时钟 | 100ms 周期脉冲 |
| M8040 | 禁止状态转移 | 暂停所有步进 |
| M8046 | STL 动作中 | 任一步为ON时为1 |

## 5.4 基本编程

### GX Works3 操作界面

```
┌─────────────────────────────────────────────┐
│  工程  编辑  显示  查询  工具  在线  诊断  帮助 │
├─────────────────────────────────────────────┤
│ [工程树]                    [工具条]          │
│  ├─ 工程                     📁 💾 🔍 🔄    │
│  │   ├─ 程序体                                  │
│  │   │   └─ P11 (MAIN)    ← 当前编辑画面        │
│  │   └─ 参数                                    │
│  └─ 软元件注释                                  │
├─────────────────────────────────────────────┤
│                                              │
│    X0 ─┤ ├──────( Y0 )                       │
│         │                                     │
│    X1 ──┤                                      │
│                                              │
│  [工具箱]                                     │
│   ├─ 常开触点  [  ]                           │
│   ├─ 常闭触点  [\/]                           │
│   ├─ 线圈      (  )                           │
│   ├─ 功能     [FUN]                           │
│   └─ 注释                                     │
└─────────────────────────────────────────────┘
```

### 示例 1：启保停控制

**要求**：X0 启动、X1 停止，控制 Y0 输出。

```
       X0         X1        Y0
  ──[ ]────────[ ]─┤──────( )──  ← 自锁
                      ↑
                   Y0 的常开触点
```

### 示例 2：顺序控制（STL 指令）

三菱独有 **STL（步进顺控）** 指令，适合顺序动作控制。

**要求**：三个灯依次点亮，每个持续 2 秒，循环。

```
    ──[ M8002 ]───────────────────────────[ SET S0 ]
              ↑ 初始脉冲，复位到 S0
    
    STL S0:
    ──[ X0 启动 ]──[ SET S21 ]───────( Y0 )──
         │                                  │
    ─────────────────────────[ RST S0 ]─────┘
    
    STL S21:   ← Y0 亮 2 秒后转 S22
    ──[ T0 K20 ]──[ SET S22 ]──( Y1 )──
         │                      │
    ──[ OUT T0 K20 ]──[ RST S21 ]──
         ↑ 定时器，T0 = 2s
    
    STL S22:   ← Y1 亮 2 秒后转 S23
    ...
```

### 示例 3：计时器和计数器组合

**要求**：生产线计数，每 100 件输出一个脉冲信号。

```
  传感器信号(X0) ──[CNT C0 K100]──( Y0 )
                     PV=100
                     计数到达输出 Q

  M8013 ──[ 1s 时钟 ]──[ RST C0 ]──
                              ↑
                         每秒复位重新计数
```

## 5.5 功能块（FB）和函数（FUN）

### 创建功能块

1. 在工程树右键 → **新建数据** → 选择 **功能块**
2. 选择语言类型：梯形图 / ST / FBD
3. 编写功能块内部逻辑

### 功能块封装示例

**功能块：Motor_Control**

```
输入：
  - start (BOOL)  启动
  - stop (BOOL)   停止
  - fault (BOOL)  故障输入

输出：
  - running (BOOL)  运行中
  - fault_out (BOOL) 故障输出

内部：
  - timer (TON)
```

**调用功能块**：

```
  [ Motor_Control  FB  ]
  ── start ─┤           ├─ running ── Y0
            ├─ stop ────┤
            ├─ fault ───┼─ fault_out
```

## 5.6 模拟量与通信

### FX5U 模拟量输入

FX5U 内置 2 路模拟量输入（通道 1 和 2），无需扩展模块：

| 通道 | 输入电压 | 分辨率 |
|------|----------|--------|
| AD1 | DC 0~10V | 12bit (0~4000) |
| AD2 | DC 0~10V | 12bit (0~4000) |

**编程读取**：

```
  [ FROM K0 H0 D0 K1 ]
           │ │  │  │  └─ 读取1点
           │ │  │  └─ 存入 D0
           │ │  └─ 读取地址 H0（CH1）
           │ └─ 特殊模块号 0
           └─ FROM指令：从扩展模块读取
```

### 以太网通信

FX5U 内置以太网口，支持以下协议：

| 功能 | 说明 |
|------|------|
| **SLMP** | 三菱专用协议，可与 HMI/SCADA 通信 |
| **Socket** | TCP/UDP 自由通信 |
| **Modbus TCP** | 标准化通信 |

**简单通信示例**：

```st
// FX5U 做 Modbus TCP 从站
// 使用 Modbus_UDP 或 Modbus_TCP 功能块
```

## 5.7 程序下载与调试

### 下载程序

1. 用 USB 或网线连接 PC 和 PLC
2. 点击 **在线** → **写入至可编程控制器**
3. 选择 **全选**（参数+程序）→ 点击 **执行**
4. 写入完成后将 PLC **切换至 RUN 模式**

### 在线监控

1. 点击 **在线** → **监视** → **监视模式**
2. 可以看到每个触点/线圈的实时状态
3. 在 **软元件登录监视** 中批量监视变量

### 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `F2` | 变换（编译）当前程序 |
| `F3` | 监视开始/停止 |
| `Shift + F3` | 强制 ON/OFF |
| `Ctrl + F7` | 写入至 PLC |
| `Ctrl + Shift + F7` | 读取从 PLC |

## 5.8 与西门子对比

| 特性 | 西门子 S7-1200 | 三菱 FX5U |
|------|---------------|-----------|
| 编程软件 | TIA Portal V17 | GX Works3 |
| 程序组织 | OB1/FB/FC/DB | Program/FB/FUN |
| 变量体系 | 符号地址 + DB | 软元件（X/Y/M/D/T/C） |
| 定时器 | TON/TOF/TONR | TON/TOF/TMR |
| 计数器 | CTU/CTD/CTUD | +C/-C |
| 通信口 | PROFINET + RS485 | 内置以太网 + RS485 |
| 模拟量 | 需要扩展模块 | 内置 2 路 ADC |
| 价格 | 较高 | 中等 |
