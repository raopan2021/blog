# 04-西门子 S7-1200 快速入门

## 4.1 软件与硬件准备

### 编程软件

西门子 PLC 使用 **TIA Portal**（全称：Totally Integrated Automation Portal）进行编程。目前主流版本为 **V17** 及以上。

下载地址：[西门子工业支持中心](https://support.industry.siemens.com/)

> ⚠️ TIA Portal 体积较大（40GB+），安装需要较高配置电脑，建议最低 16GB 内存。

### 硬件准备

| 物品 | 说明 |
|------|------|
| S7-1200 CPU（如 CPU 1214C DC/DC/DC） | 主流型号 |
| 编程线缆（USB 或以太网） | 推荐用网线，下载更快 |
| DC 24V 电源 | 为 PLC 供电 |
| 输入信号源（按钮、传感器） | 调试用 |
| 输出负载（指示灯、继电器） | 调试用 |

### S7-1200 面板说明

```
┌─────────────────────────────────┐
│  ⚡  ▼ CPU 1214C                 │

```

## 4.2 TIA Portal 项目创建

### 步骤 1：新建项目

1. 打开 TIA Portal → 点击 **"创建新项目"**
2. 填写项目名称、存储路径、作者
3. 点击 **"创建"** 按钮

### 步骤 2：添加设备

1. 点击左侧 **"项目和方案"** → **"添加新设备"**
2. 选择 **"控制器"** → 展开 **SIMATIC S7-1200** → 选择具体型号
3. 点击 **"确定"**

### 步骤 3：配置 CPU

1. 双击 CPU 进入设备配置
2. 设置 **PROFINET 接口** IP 地址（如 192.168.0.10）
3. 在 **常规** → **高级选项** 中开启高速计数器、脉冲输出等功能

### 步骤 4：编写程序

1. 展开 CPU → 双击 **"程序循环"** 或 **"Main[OB1]"**
2. 在打开的编程界面拖拽触点和线圈进行编程
3. 也可以在左侧项目树 → **"PLC 程序"** → **"添加新块"** 创建功能块（FB）或函数（FC）

## 4.3 变量与地址

### 绝对地址

| 地址 | 类型 | 说明 |
|------|------|------|
| I0.0 ~ I1.5 | 输入（Input） | 数字量输入 |
| Q0.0 ~ Q1.5 | 输出（Output） | 数字量输出 |
| M0.0 ~ | 中间标志位（Marker） | 内部继电器 |
| DB1.DBX0.0 | 数据块位 | 全局数据块中的位 |
| IW0 / QW0 | 字 | 模拟量输入/输出 |

### 符号地址（推荐）

在 **PLC 变量** 页面定义符号名：

| 符号名 | 地址 | 数据类型 | 说明 |
|--------|------|----------|------|
| start_btn | I0.0 | Bool | 启动按钮 |
| stop_btn | I0.1 | Bool | 停止按钮 |
| motor_run | Q0.0 | Bool | 电机运行输出 |
| counter | MW0 | Int | 计数器 |

> 💡 推荐用符号名编程，程序可读性更强，改硬件时只需改变量表。

## 4.4 基本编程示例

### 示例 1：启保停控制

**要求**：按下启动按钮 I0.0，Q0.0 输出；按下停止按钮 I0.1，Q0.0 停止。

**梯形图**：

```
  start_btn      stop_btn        motor_run
  I0.0            I0.1             Q0.0
   │               ││               │
──[ ]─────────────[ ]─┤───────────( )──
                      ↑
                  motor_run 的自锁触点
```

**等效 SCL 代码**：

```st
// 启保停逻辑
motor_run := (start_btn OR motor_run) AND NOT stop_btn;
```

### 示例 2：延时启停

**要求**：启动信号来后 5 秒电机启动，停止信号来后 3 秒电机停止。

**编程方法**：使用定时器

1. 创建 **IEC Timer** 类型变量 `ton1`
2. 梯形图：

```
  start_sig ──[TON IN]──( )── Q0.0
              PT:=T#5S
              ET:=当前值

  stop_sig ──[TOF IN]──( )── Q0.0
             PT:=T#3S
```

3. SCL 版本：

```st
ton1(IN := start_sig, PT := T#5S);
Q0.0 := ton1.Q;

tof1(IN := stop_sig, PT := T#3S);
Q0.0 := Q0.0 AND NOT tof1.Q;
```

### 示例 3：计数控制

**要求**：每按下 10 次按钮，指示灯亮一次。

```
  btn ──[CTU CU]──( )─ 指示灯
          PV:=10
          Q=计数到达
```

```st
// 按下按钮时增计数
IF btn_edge AND NOT last_btn THEN
    counter := counter + 1;
END_IF;
last_btn := btn;

// 到达 10 次后复位计数器并点亮灯
IF counter >= 10 THEN
    counter := 0;
    lamp := TRUE;
END_IF;
```

## 4.5 程序下载与调试

### 下载程序

1. 用网线连接 PC 和 PLC，将 PC 网卡 IP 设为 **同一网段**（如 192.168.0.100）
2. 在 TIA Portal 中点击 **"下载到设备"** 按钮 🔻
3. 选择 PG/PC 接口类型为 **PN/IE** → 双击找到对应 PLC
4. 选择 **"显示所有可访问的设备"** → 点击 **"装载"**

### 在线监控

1. 点击 **"转到在线"** 🔗 或按 `Ctrl+L`
2. 点击 **"监视"** 👁️ 按钮，可以实时看到触点通断和数值变化
3. 在程序上 **右键 → 监视 → 强制** 可以强制修改变量值进行调试

### 常见下载问题

| 问题 | 解决方法 |
|------|----------|
| 找不到设备 | 检查网线连接、IP 网段、Ping 测试 |
| CPU 需要重启 | 将 CPU 拨到 STOP，再拨回 RUN |
| 密码保护 | 联系程序作者获取密码 |

## 4.6 S7-1200 高速功能

### 高速计数器（HSC）

用于统计编码器脉冲，计数频率可达 100kHz+。

**配置步骤**：
1. 在 CPU 属性 → **高速计数器** → 启用 HSC
2. 选择计数类型（频率、周期、脉冲计数）
3. 设置初始值和预设值
4. 在程序中读取计数值

```st
// 读取 HSC1 的计数值
hsc_count := "HSC1_DB".cnt;
```

### 脉冲输出（PTO/PWM）

用于控制步进电机或伺服驱动器。

1. 在 CPU 属性 → **脉冲发生器** → 启用 PTO/PWM
2. 分配 Q 脉冲输出点（如 Q0.0）
3. 使用 **Motion Control** 指令进行位置控制

## 4.7 数据块（DB）使用

数据块用于存储大量数据，类似变量集合。

**创建数据块**：
1. 右键 **PLC 程序** → **添加新块** → 选择 **数据块（DB）**
2. 命名如 `Data_Recipe`

**数据块内容示例**：

| 名称 | 数据类型 | 起始值 | 说明 |
|------|----------|--------|------|
| motor_speed | Int | 1500 | 电机转速 RPM |
| belt_length | Real | 10.5 | 传送带长度 m |
| recipe_1 | Array[1..10] of Int | - | 配方数组 |

**访问数据块**：

```st
// 读取
speed := "Data_Recipe".motor_speed;

// 写入
"Data_Recipe".motor_speed := 2000;
```
