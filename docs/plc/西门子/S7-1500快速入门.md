# S7-1500 快速入门

## S7-1500 vs S7-1200

| 项目 | S7-1200 | S7-1500 |
|------|---------|---------|
| 定位 | 中小型 | 中大型 |
| 程序容量 | 最大 1MB | 最大 4MB |
| 执行速度 | 快 | 更快（同等条件约 3 倍） |
| 扩展能力 | 最多 1 个 CM | 最多 4 个 CM |
| 运动控制 | 基础 PTO | 高级 Motion Control |
| 安全功能 | 基础 | 集成 F-CPU（故障安全） |
| 通信接口 | 1×PN + 可选 | 2×PN + 可选 |
| 诊断 | 基础 LED | 彩色 LED + 显示屏 |

## TIA Portal 配置差异

S7-1500 的编程方式与 S7-1200 基本相同，以下为主要差异点：

### 1. 硬件配置

1. 添加设备时选择 **SIMATIC S7-1500** → 选择具体 CPU（如 CPU 1515-2 PN）
2. S7-1500 支持 **多 Profinet 接口**，可分别配置为不同网段
3. 可选配 **工艺模块**（TM）用于计数、位置等

### 2..Trace（trace 示波器）功能

S7-1500 独有 **Trace** 功能，无需额外软件即可记录变量波形：

1. 在 CPU 上右键 → **"启动 Trace"**
2. 选择要记录的变量（最多 16 个）
3. 设置触发条件和采样周期
4. 记录完成后在趋势图中查看波形

```
Trace 用途：
- 观察模拟量波动
- 调试 PID 参数
- 分析故障时的信号变化
```

### 3. 工艺对象（Technological Objects）

S7-1500 支持更完善的工艺对象：

| 工艺对象 | 说明 |
|----------|------|
| **轴 (Axis)** | 步进/伺服轴，支持速度、位置、扭矩控制 |
| **凸轮 (Cam)** | 复杂运动曲线定义 |
| **同步动画 (Kinematics)** | 多轴联动（机器人等） |
| **测量输入 (Measuring Input)** | 高速信号采集 |

### 4. 运动控制示例（SCL）

::: code-group
```txt [st]
// 位置控制
"Axis_1".MoveAbsolute(
    Execute  := start_move,
    Position := target_pos,   // 目标位置（mm）
    Velocity := max_speed,   // 最大速度（mm/s）
    Done     := move_done,
    Error    := move_error
);

// 停止
"Axis_1".MoveStop(
    Execute := emergency_stop
);
```
:::

### 5. S7-1500 报警系统

S7-1500 支持 **程序化的报警（Program Alarms）**：

::: code-group
```txt [st]
// 编程方式生成诊断报警
"Alarm_1"(Alarm := TRUE,
    Issue := "温度超限，当前温度：" + INT_TO_STRING(temperature));
```
:::

## 安全 PLC（F-CPU）

S7-1500 提供 **F 系列（F-CPU）**，内置故障安全功能：

- 安全 I/O 模块直接连接
- 安全程序与标准程序分离（独立安全 OB）
- 支持 PROFIsafe 安全协议

::: code-group
```txt [st]
// F-CPU 安全程序示例（F-LAD）
// 急停监控
  emergency_stop ──[ F-IORED ]──( )── 安全输出
```
:::

## 选型建议

| 应用规模 | 推荐型号 |
|----------|----------|
| 小型设备（<100 IO） | S7-1200 CPU 1214C |
| 中型设备（100~500 IO） | S7-1500 CPU 1513C |
| 大型设备（500+ IO） | S7-1500 CPU 1515C 及以上 |
| 高速运动控制 | S7-1500 T + 工艺模块 |
| 安全相关应用 | S7-1500 F-CPU |
