# 03-编程语言基础

PLC 提供了多种编程语言，国际标准 IEC 61131-3 定义了以下五种：

| 语言类型 | 缩写 | 特点 | 适合人群 |
|----------|------|------|----------|
| **梯形图** | LD / LAD | 可视化，类似电气原理图 | 电气工程师 |
| **功能块图** | FBD | 由功能块连线构成 | 过程控制工程师 |
| **结构化文本** | ST | 类似 Pascal/C 代码 | 软件工程师 |
| **指令表** | IL | 助记符文本（已逐步淘汰） | 老式 PLC 用户 |
| **顺序功能图** | SFC | 步进式流程描述 | 复杂顺序控制 |

本教程重点讲解 **梯形图（LAD）** 和 **结构化文本（ST）**。

## 1. 梯形图（Ladder Diagram, LAD）

梯形图是 PLC 最通用、最直观的编程语言，源自继电器控制电路图。左侧为"火线"，右侧为"零线"，电流从左流向右。

### 基本符号

| 符号 | 名称 | 功能 | 等效电路 |
|------|------|------|----------|
| `-[ ]-` | 常开触点 | 条件接通时导通 | 按钮（不按） |
| `-[\/]-` | 常闭触点 | 条件断开时导通 | 按钮（按下） |
| `-( )-` | 线圈输出 | 条件成立则输出 | 接触器线圈 |
| `-(S)-` | 置位线圈 | 接通后保持 | 自锁按钮 |
| `-(R)-` | 复位线圈 | 断开输出 | 停止按钮 |
| `-[F]-` 或 `-[TON]-` | 功能/定时器 | 调用功能块 | - |

### 经典电路

**① 自锁电路（启保停）**

按下启动按钮 → 电机运转 → 松开按钮后保持运行 → 按下停止按钮 → 电机停止

```
        I0.0(启动)         I0.1(停止)
          │                 │┐
          │                 ││  ← 常闭
          │    ┌─────────────┘│
          │    │              │
    ──[ ]─┤──[ ]─┤─────────( )─ Q0.0  电机
              ↑
           自锁触点（Q0.0 的常开）
```

**② 互锁电路**

两个接触器不能同时吸合（如正反转控制）：

```
        I0.0(正转)                        I0.1(反转)
          │                                  │
          │    ┌──────────────────────────[ ]─┤
          │    │                              │
    ──[ ]─┤──[ ]─┤───────────────────────(S)─ Q0.0  正转接触器
              ↑                              │
           Q0.1 的常闭（反转吸合时断开正转）    │
                                              │
          I0.1(反转)   ┌──────────────────────┘
            │          │
            │    ┌─────┴─────┐
            │    │  (自复位)  │
            └────┤           │
                 └───────────┘
```

### 编程规则

1. **从上到下、从左到右** 扫描执行
2. **同一输出线圈不能重复使用**（同一 Q 点多次输出以最后一次为准）
3. **触点可以串联也可以并联**，线圈输出只能放在最右侧
4. **梯形图左侧的垂直线** 相当于电源火线，**右侧** 相当于零线

## 2. 结构化文本（Structured Text, ST）

ST 是一种高级文本语言，类似 Pascal/C，适合复杂算法和数学运算。

### 基本语法

```st
// 变量定义
VAR
    counter : INT := 0;      // 计数器
    start_flag : BOOL;        // 启动标志
END_VAR

// 赋值
counter := counter + 1;

// 条件语句
IF counter > 100 THEN
    counter := 0;
    start_flag := TRUE;
ELSIF counter < 0 THEN
    counter := 0;
ELSE
    start_flag := FALSE;
END_IF;

// 循环（少用，PLC 中一般不用循环）
FOR i := 0 TO 10 DO
    sum := sum + i;
END_FOR;

// CASE 多分支
CASE status OF
    0:  motor_on := FALSE;
    1:  motor_on := TRUE;
    2:  alarm := TRUE;
ELSE
    alarm := TRUE;
END_CASE;
```

### 西门子 SCL（ST 方言）示例

```st
// S7-1200/S7-1500 中使用 SCL
#counter := #counter + 1;

IF #counter >= 100 THEN
    #counter := 0;
    #done := TRUE;
END_IF;

// 布尔表达式
#motor_run := #start AND NOT #stop AND NOT #fault;
```

## 3. 功能块图（FBD）

通过功能块之间的连线传递信号，适合信号流清晰的控制系统。

```
  [启动按钮]────[  ≥1  ]────[  RS  ]────[  Q0.0  ]
       │              ▲       │
  [停止按钮]──────[  NOT ]─────┘
```

## 4. 定时器和计数器

### 定时器

| 定时器类型 | 西门子（S7） | 三菱（FX） | 功能 |
|-----------|-------------|-----------|------|
| 通电延时 | TON | TON | 输入ON，延迟输出 |
| 断电延时 | TOF | TOF | 输入OFF，延迟输出 |
| 保持型延时 | TONR | TMR | 累积多次接通时间 |

**TON（通电延时）示例**：
```
  I0.0 ──[TON T0]──( )─ Q0.0
          PT=5s
          ET=当前值
```
- I0.0 接通 → 定时器开始计时 → 5秒后 Q0.0 输出
- I0.0 断开 → Q0.0 立即复位，计时值清零

### 计数器

| 计数器类型 | 西门斯 | 三菱 | 功能 |
|-----------|--------|------|------|
| 增计数器 | CTU | +C | 到达设定值输出 |
| 减计数器 | CTD | -C | 减到0输出 |
| 增减计数器 | CTUD | -C/+C | 可增可减 |

```
  I0.0(加) ──[CTU C0]──( )─ Q0.0
                PV=5
                Q=计数到达
```

## 5. 数据类型

| 数据类型 | 大小 | 示例 | 说明 |
|----------|------|------|------|
| BOOL | 1 bit | TRUE / FALSE | 开关量 |
| INT | 16 bit | 32767 | 整数 |
| DINT | 32 bit | 2147483647 | 双整数 |
| REAL | 32 bit | 3.14159 | 浮点数 |
| TIME | 32 bit | T#5S | 时间类型 |
| STRING | N byte | 'Hello' | 字符串 |

## 编程习惯

1. **变量命名规范**：用有意义的名称，如 `motor_start_btn`、`conveyor_speed` 而非 `I0.0`、`M10`
2. **程序分段**：手动程序、自动程序、报警程序分块编写
3. **注释完整**：每个程序段的功能、作者、日期都要标注
4. **巧用标志位**：用中间继电器（M）解耦逻辑，避免双线圈问题
