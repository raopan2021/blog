# PLC 编程入门

可编程逻辑控制器（PLC）是工业自动化控制的核心设备。本教程从基础概念出发，逐步深入，涵盖主流品牌（西门子、三菱）的实战编程方法。

## 📖 教程目录

### 基础入门

- [01-认识PLC](/plc/01-认识PLC) — PLC 组成、工作原理、应用场景
- [02-PLC硬件基础](/plc/02-PLC硬件基础) — CPU、模块选型、接线方式
- [03-编程语言基础](/plc/03-编程语言基础) — 梯形图（LAD）、功能块图（FBD）、结构化文本（ST）

### 进阶应用

- [04-西门子S7-1200快速入门](/plc/04-西门子S7-1200快速入门) — TIA Portal 环境搭建、博途使用
- [05-三菱FX5U快速入门](/plc/05-三菱FX5U快速入门) — GX Works3 环境搭建
- [06-常用功能与实战](/plc/06-常用功能与实战) — 计数器、定时器、模拟量、通信

## 💡 前置知识

- 基本的电路知识（直流/交流、控制回路）
- 了解继电器、接触器的工作原理
- 有任何一门编程语言基础更佳（非必须）

> 本教程以 **西门子 S7-1200** 和 **三菱 FX5U** 为主要讲解品牌，其他品牌触类旁通。
