# PLC 工业自动化

可编程逻辑控制器（PLC）是工业自动化控制的核心设备。本教程从基础概念出发，逐步深入，涵盖主流品牌（西门子、三菱、ABB）的实战编程方法。

## 💡 前置知识

- 基本的电路知识（直流/交流、控制回路）
- 了解继电器、接触器的工作原理
- 有任何一门编程语言基础更佳（非必须）

> 本教程以 **西门子 S7-1200**、**三菱 FX5U** 和 **ABB AC500** 为主要讲解品牌，其他品牌触类旁通。
