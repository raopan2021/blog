# 梯形图基础

梯形图（Ladder Diagram, LAD）是 PLC 最通用、最直观的编程语言，源自继电器控制电路图。左侧为"火线"，右侧为"零线"，电流从左流向右。

## 基本符号

| 符号 | 名称 | 功能 | 等效电路 |
|------|------|------|----------|
| `-[ ]-` | 常开触点 | 条件接通时导通 | 按钮（不按） |
| `-[\/]-` | 常闭触点 | 条件断开时导通 | 按钮（按下） |
| `-( )-` | 线圈输出 | 条件成立则输出 | 接触器线圈 |
| `-(S)-` | 置位线圈 | 接通后保持 | 自锁按钮 |
| `-(R)-` | 复位线圈 | 断开输出 | 停止按钮 |
| `-[F]-` 或 `-[TON]-` | 功能/定时器 | 调用功能块 | - |

## 经典电路

### ① 自锁电路（启保停）

按下启动按钮 → 电机运转 → 松开按钮后保持运行 → 按下停止按钮 → 电机停止

```
        I0.0(启动)         I0.1(停止)
          │                 │┐
          │                 ││  ← 常闭
          │    ┌─────────────┘│
          │    │              │
    ──[ ]─┤──[ ]─┤─────────( )─ Q0.0  电机
              ↑
           自锁触点（Q0.0 的常开）
```

### ② 互锁电路

两个接触器不能同时吸合（如正反转控制）：

```
        I0.0(正转)                        I0.1(反转)
          │                                  │
          │    ┌──────────────────────────[ ]─┤
          │    │                              │
    ──[ ]─┤──[ ]─┤───────────────────────(S)─ Q0.0  正转接触器
              ↑                              │
           Q0.1 的常闭（反转吸合时断开正转）    │
                                              │
          I0.1(反转)   ┌──────────────────────┘
            │          │
            │    ┌─────┴─────┐
            │    │  (自复位)  │
            └────┤           │
                 └───────────┘
```

### ③ 闪烁电路（报警灯）

```
  M0.0 ──[ T1 常开 ]──( S )─ M0.0    ← 自锁
              ↑
         T1 的常闭触点

  M0.0 ──[ T1 ]──( )─ T1
             ↑       ↑
         T1 定时器  T1 复位触点（自复位）
```

## 编程规则

1. **从上到下、从左到右** 扫描执行
2. **同一输出线圈不能重复使用**（同一 Q 点多次输出以最后一次为准）
3. **触点可以串联也可以并联**，线圈输出只能放在最右侧
4. **梯形图左侧的垂直线** 相当于电源火线，**右侧** 相当于零线
5. **常闭触点用于停止/保护**，常开触点用于启动/使能
