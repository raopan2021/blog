# PLC 硬件组成

## PLC 硬件系统架构

PLC 系统分为 **紧凑型** 和 **模块化** 两大类：

### 紧凑型 PLC（一体机）

CPU、电源、输入输出全部集成在一个机体内，适合小型设备。代表型号：

- 三菱 **FX3U** / **FX5U**
- 西门子 **S7-200 SMART**
- 欧姆龙 **CP1E**
- ABB **AC500-eCo**

```
┌─────────────────────────┐
│   FX5U 紧凑型 PLC        │
│  ┌───────────────────┐  │
│  │ CPU │ 输入 │ 输出  │  │
│  │ 💡  │ 🔘  │ ⚡  │  │
│  └───────────────────┘  │
└─────────────────────────┘
```

### 模块化 PLC（组合式）

各功能模块独立，按需组合，适合中大型项目。代表型号：

- 西门子 **S7-1200** / **S7-1500**
- 三菱 **Q 系列**
- ABB **AC500**

```
┌────────┬────────┬────────┬────────┐
│  电源   │  CPU   │  DI    │  DO    │
│  PS    │  CPU   │  16点  │  16点  │
├────────┼────────┼────────┼────────┤
│        │  AI    │  AO    │  COM   │
│        │  4路   │  2路   │  RS485 │
└────────┴────────┴────────┴────────┘
```

## CPU 模块

CPU 是 PLC 的核心，负责：
- 执行用户程序
- 处理输入输出数据
- 管理通信
- 监控系统运行状态

### 关键参数

| 参数 | 含义 | 选型建议 |
|------|------|----------|
| 程序容量 | 用户程序最大存储空间 | 根据项目复杂度选择，一般 50KB 起 |
| 布尔执行速度 | 每秒执行布尔指令数 | 高速设备选 100KB/s 以上 |
| 定时器/计数器数量 | 内置资源数量 | 注意规格书中软元件数量上限 |
| 通信接口 | 编程/设备联网接口 | 至少需要 1 个编程口 + 网口更佳 |
| 供电电压 | 24VDC 或 220VAC | 根据现场条件选择，24VDC 更安全 |

### 指示灯含义（各品牌通用）

| LED 状态 | 含义 |
|----------|------|
| RUN（绿色常亮） | 程序正常运行中 |
| RUN（绿色闪烁） | CPU 运行但有警告 |
| STOP（黄色/橙色） | 程序停止或处于停机模式 |
| ERROR/FAULT（红色） | 严重故障，需排查 |
| COM（闪烁） | 正在进行通信 |

## 扩展模块

| 模块类型 | 标记 | 功能 |
|----------|------|------|
| 数字量输入 | DI | 增加数字输入点数 |
| 数字量输出 | DO | 增加数字输出点数 |
| 模拟量输入 | AI | 将温度/压力等转为数字量 |
| 模拟量输出 | AQ | 输出电压/电流控制变频器 |
| 通信模块 | CM | RS485 / CAN / 以太网 |
| 高速计数 | HSC | 接收编码器脉冲信号 |
| 位置控制 | PTO/PWM | 脉冲输出控制步进/伺服 |

## 硬件选型原则

1. **IO 点数留余量**：选型时实际需要点数的 **120~150%**
2. **模块化优先**：中大型项目选模块化，方便扩展和维护
3. **品牌统一**：同一项目尽量使用同一品牌，保证兼容性
4. **供电方式**：现场有 DC 24V 就选 DC 型，有 AC 220V 就选 AC 型
5. **环境因素**：高温、粉尘、震动环境下需选防护等级更高的机箱
