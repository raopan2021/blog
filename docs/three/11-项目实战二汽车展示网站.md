## 核心原理

### 1. 加载屏幕 + 进场动画

参考 su7-replica，加载阶段显示一个动画 "LOADING..."，然后播放进场动画：

```js
// 加载屏幕 DOM
document.querySelector('#app').innerHTML = `
  <div class="loader-screen">
    <div class="loading">
      <span style="--i:0">L</span>
      <span style="--i:1">O</span>
      <span style="--i:2">A</span>
      ...
    </div>
  </div>
`

// 3秒后淡出并播放进场
setTimeout(() => {
  loaderScreen.classList.add('hollow') // CSS 淡出
  playIntroAnimation()
}, 3000)
```

CSS 动画让字母依次「跳动」：

```css
.loading span {
  animation: loadingPulse 1.5s ease-in-out infinite;
  animation-delay: calc(var(--i) * 0.1s); /* 每个字母延迟0.1秒 */
}
@keyframes loadingPulse {
  0%, 100% { opacity: 0.3; transform: scale(0.95); }
  50% { opacity: 1; transform: scale(1.05); }
}
```

### 2. GSAP 时间线动画

使用 GSAP 的 timeline 协调多个动画同时进行（类似视频编辑软件的时间线）：

```js
import gsap from 'gsap'

function playIntroAnimation() {
  // 时间线：相机推进 + 灯光渐亮 + 环境变化 同步进行
  const tl = gsap.timeline()

  // 1. 相机从远处推进到车身侧面
  tl.to(camera.position, {
    x: 0, y: 0.8, z: -7,
    duration: 4,
    ease: 'power2.inOut',
  })

  // 2. 灯光强度渐强（延迟1秒后开始）
  tl.to(roomLights, {
    intensity: (i) => roomLights[i].userData.originalIntensity,
    duration: 4,
    delay: 1,
    ease: 'power2.inOut',
    // 每盏灯错开0.2秒
    stagger: 0.2,
  })

  // 3. 环境雾浓度变化
  tl.to(scene.fog, {
    near: 15, far: 60,
    duration: 4,
    delay: 0.5,
  }, '<') // '<' 表示与上一个动画同时开始
}
```

**timeline 的好处**：多个动画可以精确对齐、相对定位（`'-=1'` 提前1秒，`'<'` 同时开始），比手动用 `setTimeout` 精确得多。

### 3. 点击触发加速模式

通过射线检测判断是否点击到汽车：

```js
const raycaster = new THREE.Raycaster()

window.addEventListener('click', (event) => {
  // 归一化鼠标坐标
  mouse.x = (event.clientX / window.innerWidth) * 2 - 1
  mouse.y = -(event.clientY / window.innerHeight) * 2 + 1

  // 发射射线检测
  raycaster.setFromCamera(mouse, camera)
  const intersects = raycaster.intersectObjects(carGroup.children, true)

  if (intersects.length > 0) {
    triggerRush()
  }
})
```

### 4. Speed Lines 着色器

使用自定义 `ShaderMaterial` 在屏幕空间绘制速度线，模拟赛车游戏中的「残影」效果：

```js
const speedLinesMesh = new THREE.Mesh(
  new THREE.PlaneGeometry(2, 2),  // 全屏平面
  new THREE.ShaderMaterial({
    transparent: true,
    depthWrite: false,
    uniforms: {
      uSpeed: { value: 0.0 },    // 当前速度 0~1
      uOpacity: { value: 0.0 },  // 透明度
      uTime: { value: 0.0 },     // 时间（用于动画）
    },
    vertexShader: `
      varying vec2 vUv;
      void main() {
        vUv = uv;
        // 直接输出 NDC 坐标，覆盖整个屏幕
        gl_Position = vec4(position, 1.0);
      }
    `,
    fragmentShader: `
      uniform float uSpeed;
      uniform float uOpacity;
      uniform float uTime;
      varying vec2 vUv;

      float random(vec2 st) {
        return fract(sin(dot(st.xy, vec2(12.9898, 78.233))) * 43758.5453);
      }

      void main() {
        if (uSpeed < 0.01) {
          gl_FragColor = vec4(0.0);
          return;
        }

        // 中心点（消失点）
        vec2 center = vec2(0.5, 0.35);
        vec2 dir = vUv - center;
        float dist = length(dir);
        float angle = atan(dir.y, dir.x);

        // 沿径向绘制线条（极坐标 + 正弦波）
        float speedLine = sin(angle * 30.0 + uTime * 10.0) * 0.5 + 0.5;

        // 远处细、近处粗
        float thickness = 0.002 * (1.0 - dist * 0.5);

        // 中心亮、边缘暗
        float radialFade = smoothstep(0.8, 0.0, dist);

        // 随机线条
        float noise = random(vec2(angle * 0.1, floor(uTime * 5.0)));

        float line = speedLine * radialFade * noise * uSpeed;
        line = smoothstep(thickness, thickness + 0.01, line);

        // 青色渐变
        vec3 color = mix(vec3(1.0), vec3(0.0, 0.96, 1.0), dist * 0.5);
        gl_FragColor = vec4(color, line * uOpacity);
      }
    `,
  })
)

// 关键：禁用深度写入，渲染在所有物体之上
speedLinesMesh.renderOrder = 999
speedLinesMesh.frustumCulled = false
```

### 5. 加速模式的时间线编排

```js
function triggerRush() {
  // 1. 速度线淡入
  gsap.to(speedLinesMesh.material.uniforms.uOpacity, {
    value: 0.8, duration: 1, ease: 'power2.out',
  })

  // 2. 速度值上升（0 → 10）
  gsap.to({ speed: 0 }, {
    speed: 10,
    duration: 4,
    ease: 'power2.out',
    onUpdate() {
      const s = this.targets()[0].speed
      speedLinesMesh.material.uniforms.uSpeed.value = s / 10
    },
  })

  // 3. FOV 增大（广角效果 = 加速感）
  gsap.to(camera, {
    fov: 45,
    duration: 3,
    ease: 'power2.out',
    onUpdate: () => camera.updateProjectionMatrix(),
  })

  // 4. Bloom 增强
  gsap.to(bloomPass, { strength: 1.8, duration: 3 })

  // 5. 展厅灯光调暗
  emissiveMaterials.forEach((mat) => {
    gsap.to(mat, {
      emissiveIntensity: mat.emissiveIntensity * 0.3,
      duration: 2,
    })
  })
}
```

### 6. 相机抖动

在加速模式下，每帧给相机添加一个基于噪声的随机偏移：

```js
// 在动画循环中
if (isRushing) {
  const shakeIntensity = 0.05 * Math.min(speed / 10, 1)
  camera.position.x += (Math.random() - 0.5) * shakeIntensity
  camera.position.y += (Math.random() - 0.5) * shakeIntensity
}
```

真实项目中会用 Simplex Noise（FBM）让抖动更「平滑」：

```js
// Simplex 2D noise
function noise2D(x, y) { /* 省略实现 */ }
function fbm({ frequency = 2, amplitude = 0.5 }) {
  let value = 0
  for (let i = 0; i < 4; i++) {
    value += noise2D(frequency, frequency) * amplitude
    frequency *= 2
    amplitude *= 0.5
  }
  return value
}

// 相机抖动
const posOffset = new THREE.Vector3(
  fbm({ frequency: t * 0.5 }),
  fbm({ frequency: t * 0.5 + 1000 }),
  fbm({ frequency: t * 0.5 + 2000 }),
)
posOffset.multiplyScalar(0.1 * intensity)
camera.position.add(posOffset)
```

### 7. 汽车模型结构

使用多个 Three.js 几何体组合成一辆完整的汽车：

```
车身结构：
├── 车身底部（BoxGeometry，深色塑料件）
├── 车身主体（BoxGeometry，蓝色金属漆）
├── 前舱盖（倾斜的 BoxGeometry）
├── 后备箱
├── 车顶（薄 BoxGeometry）
├── A柱（倾斜的 BoxGeometry）
├── 前风挡玻璃（PlaneGeometry，MeshPhysicalMaterial）
├── 后窗玻璃
├── 侧窗玻璃 × 2
├── 前大灯（CircleGeometry，发光材质）
├── LED 日行灯带（BoxGeometry，细长发光条）
├── 尾灯 × 2 + 贯穿式灯带
├── 轮毂 × 4（轮胎 + 铝合金轮毂 + 刹车盘）
└── 车底裙边
```

玻璃材质使用 `MeshPhysicalMaterial` 模拟真实玻璃的透光和反射：

```js
const glassMaterial = new THREE.MeshPhysicalMaterial({
  color: 0x88ccff,
  transparent: true,
  opacity: 0.25,
  roughness: 0,
  metalness: 0.1,
  transmission: 0.95,  // 透光率
  ior: 1.5,           // 折射率
})
```

## 进阶扩展

### 练习 1：添加轨道控制器
加入 OrbitControls，让用户可以自由旋转视角观察汽车：

```js
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls.js'

const controls = new OrbitControls(camera, renderer.domElement)
controls.enableDamping = true
controls.target.set(0, 0.8, 0)
controls.minDistance = 3
controls.maxDistance = 15
```

### 练习 2：加载真实汽车模型
使用 `GLTFLoader` 加载外部汽车模型：

```js
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader.js'
const loader = new GLTFLoader()
loader.load('/path/to/car.glb', (gltf) => {
  scene.add(gltf.scene)
  gltf.scene.traverse((child) => {
    if (child.isMesh) child.castShadow = true
  })
})
```

### 练习 3：添加环境贴图（HDRI）
使用 HDRI 环境贴图替代简单灯光，大幅提升真实感：

```js
import { RGBELoader } from 'three/examples/jsm/loaders/RGBELoader.js'
new RGBELoader().load('/path/to/environment.hdr', (texture) => {
  texture.mapping = THREE.EquirectangularReflectionMapping
  scene.environment = texture
  scene.background = texture
})
```
