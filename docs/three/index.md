# Three.js 教程

> Three.js 是一个基于 WebGL 的 3D 图形库，用于在浏览器中创建 3D 图形

## 📚 教程目录

### 基础篇
- [01 入门：场景、相机、渲染器](./01-入门) - 创建你的第一个 3D 场景
- [02 几何体与材质](./02-几何体与材质) - 各种 3D 形状和材质
- [03 光照详解](./03-光照与阴影) - 点光源、平行光、聚光灯、阴影

### 进阶篇
- [04 纹理与贴图](./04-纹理与贴图) - 加载图片作为纹理
- [05 相机与控制器](./05-相机与控制器) - 轨道控制器、透视相机
- [06 动画](./06-动画) - 旋转、缩放、缓动动画
- [07 加载 3D 模型](./07-加载模型) - 加载 GLTF/GLB 模型

### 高级篇
- [08 后期处理](./08-后期处理) - 泛光、景深等效果
- [09 物理与交互](./09-物理与交互) - 点击选中、碰撞检测
