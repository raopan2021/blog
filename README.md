| 分类 | 说明 |

| AI 大模型 / 算法 | 前沿技术 |

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build

# 预览构建产物
pnpm preview
```

## 技术栈

- **框架**: VitePress 2.0 + Vue 3
- **样式**: SCSS + Element Plus
- **字体**: 霞鹜文楷 (LXGW WenKai Lite)
- **部署**: GitHub Pages
